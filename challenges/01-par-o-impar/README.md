# Reto 01 - Par o Impar

### Descripción

Crea una función que determine si un número entero es par o impar.

### Entrada

Un número, por ejemplo: `7`.

### Salida

Un string: `"Es par"` o `"Es impar"`.

### Ejemplo

```javascript
determinar(10) ➔ "Es par".
```

### Pista

Usa el operador de módulo `%`. Si `numero % 2` es cero, es par.
