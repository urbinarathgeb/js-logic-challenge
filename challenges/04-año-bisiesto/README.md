# Reto 04 - Año bisiesto

### Descripción

Determina si un año es bisiesto o no.

### Entrada

Un número, por ejemplo: `2024`.

### Salida

Un string: `Es bisiesto` o `No es bisiesto`.

o

Un booleano `true` o `false`.

### Ejemplo

```javascript
esBisiesto(2024) ➔ true.
```

### Pista

Un año es bisiesto si es divisible por 4 pero no por 100, o si es divisible por 400.
