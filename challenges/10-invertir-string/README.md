# Reto 10 - Invertir string

### Descripción

Crea una función que determine si un número entero es par o impar.

### Entrada

Un string, por ejemplo `Hola`.

### Salida

El string al revés.

### Ejemplo

```javascript
invertirString("Hola") ➔ "aloH".
```

### Pista

Recorre el string con un bucle for empezando por el final.
