# Reto 09 - FizzBuzz (un clásico)

### Descripción

Imprime los números del `1` al `50`, pero con reglas especiales.

### Reglas

- Si el número es divisible por `3`, imprime `Fizz`.
- Si el número es divisible por `5`, imprime `Buzz`.
- Si el número es divisible por `3` y `5`, imprime `FizzBuzz`.
- Si el número no es divisible por `3` ni `5`, imprime el número.

### Salida

La lista de resultados en consola.
