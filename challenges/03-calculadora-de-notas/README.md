# Reto 03 - Calculadora de notas

### Descripción

Convierte una puntuación numérica en una letra de calificación.

### Entrada

Un número entre `0` y `100`.

### Salida

Un string: `"A"`, `"B"`, `"C"`, `"D"`, `"F"`.

- 90 a 100: "A"
- 80 a 89: "B"
- 70 a 79: "C"
- 60 a 69: "D"
- Menos de 60: "F"

### Ejemplo

```javascript
notaFinal(95) ➔ "A".
```
