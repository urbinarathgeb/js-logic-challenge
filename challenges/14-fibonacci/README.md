# Reto 14 - Serie de Fibonacci

### Descripción

Muestra los primeros `n` números de la secuencia de Fibonacci, donde cada número es la suma de los dos anteriores.

### Entrada

Un número `n`, por ejemplo `5`.

### Salida

Un string: "1, 1, 2, 3, 5".

### Ejemplo

```javascript
fibonacci(5) ➔ "1, 1, 2, 3, 5".
```
