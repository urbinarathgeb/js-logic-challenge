# Reto 13 - Número Primo

### Descripción

Indica si un número es primo o no.

### Entrada

Un número, por ejemplo `5`.

### Salida

Un string: `Es primo` o `No es primo`.

o

Un booleano: `true` o `false`.

### Ejemplo

```javascript
esPrimo(5) ➔ "Es primo".
```

### Pista

Un número es primo si solo es divisible por 1 y por sí mismo.
