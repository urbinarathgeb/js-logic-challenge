# Reto 05 - Estaciones del año

### Descripción

Según el número del mes, indica la estación del año. (Hemisferio Sur)

### Entrada

Un número entre `1` y `12`.

### Salida

Un string: `"Verano"`, `"Otoño"`, `"Invierno"` o `"Primavera"`.

### Ejemplo

```javascript
determinar(12) ➔ "Verano".
```

### Pista

Puedes usar `switch` para agrupar los meses.
