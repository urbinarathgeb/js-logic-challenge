# Reto 08 - Contador de vocales

### Descripción

Crea una función que determine cuantas vocales tiene una palabra o frase

### Entrada

Un String, por ejemplo: `Hola Mundo`

### Salida

Un número. El total de vocales.

### Ejemplo

```javascript
contarVocales("Hola Mundo") ➔ 4
```

### Pista

Recorre el string con un `for` y usa `.toLowerCase()` para convertir el string a minúsculas.
