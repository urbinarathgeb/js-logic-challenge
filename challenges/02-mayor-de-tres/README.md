# Reto 02 - Mayor de tres

### Descripción

Dados tres números, identifica cuál es el más grande de todos.

### Entrada

Tres números, por ejemplo: `7, 10, 5`

### Salida

Un string: `El número mayor es: 10`.

### Ejemplo

```javascript
mayorDeTres(7, 10, 5) ➔ "El número mayor es: 10".
```

### Pista

Usa `if` anidados o operadores lógicos como `&&`. ¡No uses `Math.max()`!
